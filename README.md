## Quick start

goto super hero [website](https://superheroapi.com) apply token

```bash
$ git clone https://github.com/bunnywong/challenge-hero

$ cd challenge-hero

# Clone `.env.example` as `.env.development`. also replace the token collected from first step

$ npm run start
```

## Run test

```bash
$ npm run test
```
